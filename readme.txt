== Description ==
A simple plugin to display the total number of articles views by widget or/and by shortcode. You must to have [WP-PostViews](https://wordpress.org/plugins/wp-postviews/) installed

== Installation ==

1. Open `wp-content/plugins` Folder
2. Put: `Folder: the-total-views`
3. Activate `The Total Views` Plugin
4. Place the widget or shortcode

== Frequently Asked Questions ==


= How To display the total views =
1. Go to `Apparence -> Widgets -> Place the widget where you want`.
2. Type any title you like in the header widget.
3. Or type the shortcode [thetotalviews] in a page or get_thetotalviews(); in a php file.
